To print the contents of a vector in Rust, you can iterate over its elements and print each element individually. Here's an example: ```rust fn main() { // Create a vector of integers let numbers = vec![1, 2, 3, 4, 5]; // Iterate over the elements of the vector and print each element for num in &numbers { println!("{}", num); } } ```

In this example, we create a vector called `numbers` containing integers `1` through `5`. We then use a `for` loop to iterate over the elements of the vector. The `&numbers` syntax borrows the vector immutably, allowing us to iterate over its elements without taking ownership of the vector itself.

Inside the loop, we use `println!("{}", num);` to print each element of the vector. `num` represents each element in the vector as we iterate over it. When you run this code, it will print each element of the vector on a separate line: ``` 1 2 3 4 5 ``` This is how you can print the output of a vector in Rust.
